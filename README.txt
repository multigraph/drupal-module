This is the the Multigraph project for Drupal.

Multigraph is an open-source tool for creating interactive graphs in web pages.  Multigraph
is independent of Drupal and is available from www.multigraph.org.

This project contains two modules that facilitate the creation and display of Multigraph
graphs in a Drupal site:

The "Multigraph" module defines a new content type whose body contains the source XML (MUGL)
for a graph.  The normal view mode of a Multigraph node displays the graph itself, and the add/edit
form displays the MUGL source code for the graph.
